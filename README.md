# se-quick-start

Курс по основам профессиональной разработки для тех, кто умеет писать
алгоритмы на C++, но не работал инженером.

Курс концептуальный, а не синтаксический. Цель — понимать, что
происходит и почему, чтобы свободно работать с AI-агентами и судить
о результате их работы.

## Что нужно заранее

- Уверенное знание алгоритмов.
- Базовый C++ (тот, что используется на олимпиадах и в учебных
  задачах) — типы, структуры, указатели, STL.
- Установленные на машине: `git`, любая современная ОС
  (Linux, macOS, Windows с WSL).

## Как начать

1. Склонируй репозиторий и перейди в его директорию.
2. Установи [Claude Code](https://docs.claude.com/en/docs/claude-code/overview).
3. В директории курса запусти `claude`.
4. На первом запуске агент сам расскажет о формате и задаст пару
   вопросов. Дальше — твоя инициатива: пишешь "дальше", когда готов
   к следующей теме, и спрашиваешь всё, что непонятно.

## Как это устроено

- [`CURRICULUM.md`](CURRICULUM.md) — программа курса. Десять
  спринтов от установки инструментов до эксплуатации сервиса в проде.
- [`PROGRESS.md`](PROGRESS.md) — отметки о пройденных темах. Агент
  ведёт сам.
- `content/` — конспекты по темам, которые агент пишет по ходу
  занятий.
- `practice/` — место для твоего собственного кода.

Остальные файлы — служебные. Редактировать руками ничего не нужно;
агент сам поддерживает состояние.

## Как продолжить

Запусти `claude` в директории курса. Агент прочитает свои заметки
и продолжит с места, где остановились.
